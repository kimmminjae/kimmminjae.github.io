Coveo for Sitecore Hive Framework lets developers utilize the crucial functions of Coveo search. With the installation of Coveo for Sitecore, you get many components and their corresponding datasource items containing all sorts of configuration settings. Coveo for Sitecore Search Interface component is one of them. It has many settings which controls search behaviours. However, there are some functionalities which might seem trivial but are missing from the settings. In my opinion, clearing the facet when a new search query is performed is one of them.
